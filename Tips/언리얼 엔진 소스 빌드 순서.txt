1. https://github.com/EpicGames/UnrealEngine 에서 프로젝트 Fork 설정
2. Source Download하기
3. Setup.bat 파일을 실행해서 Dependencies 설치
4. GenerateProjectFiles.bat 파일을 실행해서 솔루션 생성
5. UE5.sln 솔루션 파일을 실행해서 Visual Studio에서 프로젝트 열기
6. 솔루션 구성을 DevelopmentEditor/Win64로 설정하고 UE5 프로젝트 우 클릭 -> 빌드
6-1. 경로 길이 260자 제한에 걸릴 경우 레지스트리 편집기에서 HKEY_LOCAL_MACHINE\SYSTEM\CurrentConrolSet\Control\FileSystem\LongPathsEnabled를 더블 클릭해서 0 상태로 저장
6-2. gpedit.msc를 설치하고 실행해서 관리 템플릿 -> 시스템 -> 파일 시스템 -> Win32 긴 경로 사용을 "사용" 상태로 변경
7. 시작 프로젝트를 UE5로 설정
8. F5 키를 눌러서 Debugger로 실행하여 언리얼 셰이더 컴파일 진행이 완료되고 이어서 에디터가 오픈되면 에디터를 종료
9. 솔루션에서 UnrealLightmass 빌드하기
10. 솔루션에서 UnrealMultiUserServer 빌드하기
11. 생성되어 있는 프로젝트의 uproject 파일을 우 클릭 -> Switch Unreal Engine version... 클릭하여 언리얼 엔진 소스 선택
