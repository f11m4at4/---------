1. C++ 프로젝트 생성
2. Project 폴더 -> Source -> Project.Target.cs 파일 복제 후 이름 변경([Project명]Server.Target.cs)
3. Visual Studio에서 [Project명]Server.Target.cs 를 프로젝트에 추가(Add)
4. 클래스 이름과 함수 이름을 [Project명]ServerTarget으로 변경
5. TargetType.Game을 TargetType.Server로 변경
6. 솔루션 구성을 DevelopmentServer/Win64로 변경하고 프로젝트 빌드
7. Project 폴더 -> Binaries -> Win64 로 이동
8. 실행 파일(.exe)을 복제하고 이름을 변경([Project명]Server.exe)
9. 솔루션 구성이 DevelopmentEditor/Win64로 변경하고 다시 프로젝트 빌드
10. 에디터의 프로젝트 설정 창에서 Server DefaultMap 설정
11. Windows(64-bit)로 클라이언트 패키징 하기
12. 패키징 설정에서 Build Target을 [Project명]Server로 변경 후 Windows(64-bit)로 패키징 하기
13. 패키징 파일에서 실행 파일(.exe)을 복제하고, 마우스 우 클릭하여 타겟 경로 끝에 -log 붙여주기
14. 외부 IP 접속(7777 포트)을 허용하기 위해 DefaultEngine.ini 파일에 아래 내용을 추가한다.

[OnlineSubsystem]
DefaultPlatformService=Null
